# Meu Financeiro

Dashboard simples para organizar finanças pessoais e de trabalho.

## Funcionalidades

- Separação por contexto **Pessoal** e **Trabalho**.
- Cadastro de receitas e despesas.
- Resumo automático com receitas, despesas e saldo.
- Barra de progresso para meta de economia mensal (20% das receitas).
- Histórico de movimentações.
- Gráfico de despesas por categoria.
- Persistência local via `localStorage`.

## Como usar

```bash
python3 -m http.server 8000
```

Depois abra `http://localhost:8000` no navegador.
